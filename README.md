# DA_Mini_Project
📊 **IPL Data Analytics & Match Prediction (2008–2025)**

This project analyzes historical Indian Premier League (IPL) data to uncover key insights such as team performance, toss impact, venue advantage, and player statistics. It also applies Machine Learning models like Logistic Regression and Random Forest to predict match outcomes based on past data.

🔹 Performed data cleaning, preprocessing, and feature engineering
🔹 Conducted exploratory data analysis (EDA) with visualizations
🔹 Built and compared ML models for prediction accuracy
🔹 Identified important factors influencing match results

This project demonstrates practical applications of data analytics and machine learning in sports analytics.
